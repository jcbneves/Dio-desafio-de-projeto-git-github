
	# Quais são os três tipos de objetos básicos do Git responsáveis pelo versionamento
		- BLOBS
		- TREES
		- COMMITS

	# Objetos
	blob — Um objeto blob é usado para armazenar o conteúdo de um único arquivo.
	árvore — Um objeto de árvore contém referências a outros blobs ou subárvores.
	commit — Um objeto commit contém a referência a outro objeto árvore e algumas outras informações (autor, committer etc.)
